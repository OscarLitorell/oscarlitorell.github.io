Writing numbers using words

This program writes out any number between -103,003 and 103,003 with words.

You can choose to use the short scale (where 109 is 1 billion) or the long scale (where 109 is 1 milliard). You can read more about the differences between the long and short scale here. If you choose to use the long scale the limit goes up to 106,000 instead. It's also possible to use numbers with decimals.
